input : object x $\longrightarrow$ output : class n

如何分类？

为分类训练数据（Training Data for Classification）

  • 如果把 Classification 硬当作 Regression 来做，会导致惩罚那些“太正确”的例子

  • 理想情况

    • 找到 Function (Model) 【假设是二元分类问题】

      • 输入：x
      • $f(x) \begin{cases} g(x) > 0, & \text{Output = class 1} \ else, & \text{Output = class2} \ \end{cases}$

    • 构建损失函数 $L(f) = \sum \limits_n \delta (f(x_n) \neq y_n)$ ，在 Training Data 上函数 f 得到错误结果的次数，错误越少，Loss 越小

    • 找到最佳函数：例如 Perceptron，SVM

      • 另外一个解决方法 —— Generative Model（以概率的角度看）

        • 给定一个 x ，判断它属于哪一个 class

          $P(C_1|x) = \frac {P(x|C_1)P(C_1)} {P(x|C_1)P(C_1) + P(x|C_2)P(C_2)}$

        • 可以用这个 Model 来生成一个 x ，计算某一个 x 出现的机率 $P(x) = P(x|C_1)P(C_1) + P(x|C_2)P(C_2)$，知道 x 的分布，利用其产生 x

Gaussian Distribution（高斯分布）

  • $f_{\mu,\Sigma (x) = \frac {1}{(2\pi)^{D/2}} \frac {1}{|\Sigma|^{1/2}} exp\lbrace - \frac {1}{2}(x - \mu)^T \Sigma^{-1} (x-\mu)}$

  • 输入：向量 x

  • 输出：采样 x 的概率密度

  • 函数的形状由均值 $\mu$ 和协方差矩阵 $\Sigma$ 决定

    • $\Sigma$ 相同，而 $\mu$ 不同，则概率分布最高点不同
    • $\mu$ 相同，而 $\Sigma$ 不同，概率分布最高点相同，而散的程度不同

  • 怎么找到 $\mu$ 和 $\Sigma$ ？

    • Maximum Likelihood（最大似然）

      • 高斯分布的最大似然估计 = 高斯抽样 $x_1, x_2, …, x_n$ 的概率

      • $L(\mu, \Sigma) = f{\mu, \Sigma}(x_1)f{\mu, \Sigma}(x2) \cdots f{\mu, \Sigma}(x_n)$

      • $\mu ^, \Sigma ^ = arg \; \max \limits_{\mu, \Sigma} L(\mu, \Sigma)$

        $\mu^* = \frac {1}{79} \sum \limits_{n=1}^{79}x_n$

        $\Sigma^ = \frac{1}{79} \sum \limits_{n=1}^{79}(x_n - \mu^)(x_n - \mu^*)^T$

        最有可能产生那 n 个点的 $\mu^, \Sigma^$

  • Modifying Model（修改模型）

    • Class 1 有 $\mu_1, \Sigma_1$，Class 2 有 $\mu_2, \Sigma_2$

    • 常见做法：不同 Class 共享同一个协方差矩阵 $\Sigma$ （$\Sigma$ 与input 的 feature size 的平方成正比）

    • 如果给不同的高斯分布以不同的协方差矩阵，则 Model 的参数会太多，从而导致协方差矩阵大，容易过拟合

    • 现假设，Class 1：$\mu_1, \Sigma, 79个样本$ ，Class 2：$\mu_2, \Sigma, 61个样本$，寻找 $\mu_1, \mu_2, \Sigma$

      • $L(\mu1, \mu_2, \Sigma) = f{\mu1, \Sigma}(x_1)f{\mu1, \Sigma}(x_2) \cdots f{\mu1, \Sigma}(x{79}) \times f{\mu_2, \Sigma}(x{80})f{\mu_2, \Sigma}(x{81}) \cdots f{\mu_2, \Sigma}(x{140})$
      • $\mu_1$ 是 Class 1 中 x 的均值，$\mu_2$ 是 Class 2 中 x 的均值
      • $\Sigma = \frac {79}{140} \Sigma_1 + \frac {61}{140} \Sigma_2$

      • 像右边的 Model 也称之为 Linear Model（线性模型）

概率模型三个步骤

  • 函数集（模型）
    • 不同的概率分布集合起来
    • $input \, x \longrightarrow \begin{cases} P(C_1|x) = \frac {P(x|C_1)P(C_1)} {P(x|C_1)P(C_1) + P(x|C_2)P(C_2)} \ If \, P(C_1|x) > 0.5, \, output:class \, 1 \ Otherwise, \, output: class \, 2 \ \end{cases}$
  • 评估函数的好坏
    • 找一个概率分布，能最大化地产生 data 的 likelihood（似然）
  • 找到最好的函数

Posterior Probability（后验概率）

$\begin{align} P(C_1|x) &= \frac {P(x|C_1)P(C_1)} {P(x|C_1)P(C_1) + P(x|C_2)P(C_2)} \ &= \frac {1}{1 + \frac {P(x|C_2)P(C_2)} {P(x|C_1)P(C_1)}} \ &= \frac {1}{1 + exp(-z)} & \text{$z = \ln \frac {P(x|C_1)P(C_1)}{P(x|C_2)P(C_2)}$} \ &= \sigma(z) & \text{Sigmoid Function} \end{align}$

经过化简，最终得到：$z = w^Tx + b$ ，也即 $P(C_1|x) = \sigma(w \cdot x + b)$
