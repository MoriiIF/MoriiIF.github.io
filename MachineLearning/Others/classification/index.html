如何分类？

为分类训练数据（Training Data for Classification）

  • 如果把 Classification 硬当作 Regression 来做，会导致惩罚那些“太正确”的例子

  • 理想情况

    • 找到 Function (Model) 【假设是二元分类问题】

      • 输入：x
      • f(x){g(x)>0,Output = class 1else,Output = class2f(x) \begin{cases} g(x) > 0, & \text{Output = class 1} \\ else, & \text{Output = class2} \\ \end{cases}f(x){g(x)>0,else,​Output = class 1Output = class2​

    • 构建损失函数 L(f)=∑nδ(f(xn)≠yn)L(f) = \sum \limits_n \delta (f(x_n) \neq y_n)L(f)=n∑​δ(f(xn​)=yn​) ，在 Training Data 上函数 f 得到错误结果的次数，错误越少，Loss 越小

    • 找到最佳函数：例如 Perceptron，SVM

      • 另外一个解决方法 —— Generative Model（以概率的角度看）

        • 给定一个 x ，判断它属于哪一个 class

          P(C1∣x)=P(x∣C1)P(C1)P(x∣C1)P(C1)+P(x∣C2)P(C2)P(C_1|x) = \frac {P(x|C_1)P(C_1)} {P(x|C_1)P(C_1) + P(x|C_2)P(C_2)}P(C1​∣x)=P(x∣C1​)P(C1​)+P(x∣C2​)P(C2​)P(x∣C1​)P(C1​)​

        • 可以用这个 Model 来生成一个 x ，计算某一个 x 出现的机率 P(x)=P(x∣C1)P(C1)+P(x∣C2)P(C2)P(x) = P(x|C_1)P(C_1) + P(x|C_2)P(C_2)P(x)=P(x∣C1​)P(C1​)+P(x∣C2​)P(C2​)，知道 x 的分布，利用其产生 x

Gaussian Distribution（高斯分布）

  • fμ,Σ(x)=1(2π)D/21∣Σ∣1/2exp{−12(x−μ)TΣ−1(x−μ)\Large {f_{\mu,\Sigma (x) = \frac {1}{(2\pi)^{D/2}} \frac {1}{|\Sigma|^{1/2}} exp\lbrace - \frac {1}{2}(x - \mu)^T \Sigma^{-1} (x-\mu)}}fμ,Σ(x)=(2π)D/21​∣Σ∣1/21​exp{−21​(x−μ)TΣ−1(x−μ)​

  • 输入：向量 x

  • 输出：采样 x 的概率密度

  • 函数的形状由均值 μ\muμ 和协方差矩阵 Σ\SigmaΣ 决定

    • Σ\SigmaΣ 相同，而 μ\muμ 不同，则概率分布最高点不同
    • μ\muμ 相同，而 Σ\SigmaΣ 不同，概率分布最高点相同，而散的程度不同

  • 怎么找到 μ\muμ 和 Σ\SigmaΣ ？

    • Maximum Likelihood（最大似然）

      • 高斯分布的最大似然估计 = 高斯抽样 x1,x2,...,xnx_1, x_2, ..., x_nx1​,x2​,...,xn​ 的概率

      • L(μ,Σ)=fμ,Σ(x1)fμ,Σ(x2)⋯fμ,Σ(xn)L(\mu, \Sigma) = f_{\mu, \Sigma}(x_1)f_{\mu, \Sigma}(x_2) \cdots f_{\mu, \Sigma}(x_n)L(μ,Σ)=fμ,Σ​(x1​)fμ,Σ​(x2​)⋯fμ,Σ​(xn​)

      • μ∗,Σ∗=arg  max⁡μ,ΣL(μ,Σ)\mu ^*, \Sigma ^* = arg \; \max \limits_{\mu, \Sigma} L(\mu, \Sigma)μ∗,Σ∗=argμ,Σmax​L(μ,Σ)

        μ∗=179∑n=179xn\mu^* = \frac {1}{79} \sum \limits_{n=1}^{79}x_nμ∗=791​n=1∑79​xn​

        Σ∗=179∑n=179(xn−μ∗)(xn−μ∗)T\Sigma^* = \frac{1}{79} \sum \limits_{n=1}^{79}(x_n - \mu^*)(x_n - \mu^*)^TΣ∗=791​n=1∑79​(xn​−μ∗)(xn​−μ∗)T

        最有可能产生那 n 个点的 μ∗,Σ∗\mu^*, \Sigma^*μ∗,Σ∗

  • Modifying Model（修改模型）

    • Class 1 有 μ1,Σ1\mu_1, \Sigma_1μ1​,Σ1​，Class 2 有 μ2,Σ2\mu_2, \Sigma_2μ2​,Σ2​

    • 常见做法：不同 Class 共享同一个协方差矩阵 Σ\SigmaΣ （Σ\SigmaΣ 与input 的 feature size 的平方成正比）

    • 如果给不同的高斯分布以不同的协方差矩阵，则 Model 的参数会太多，从而导致协方差矩阵大，容易过拟合

    • 现假设，Class 1：μ1,Σ\mu_1, \Sigmaμ1​,Σ ，79个样本，Class 2：μ2,Σ\mu_2, \Sigmaμ2​,Σ，61个样本，寻找 μ1,μ2,Σ\mu_1, \mu_2, \Sigmaμ1​,μ2​,Σ

      • L(μ1,μ2,Σ)=fμ1,Σ(x1)fμ1,Σ(x2)⋯fμ1,Σ(x79)×fμ2,Σ(x80)fμ2,Σ(x81)⋯fμ2,Σ(x140)L(\mu_1, \mu_2, \Sigma) = f_{\mu_1, \Sigma}(x_1)f_{\mu_1, \Sigma}(x_2) \cdots f_{\mu_1, \Sigma}(x_{79}) \times f_{\mu_2, \Sigma}(x_{80})f_{\mu_2, \Sigma}(x_{81}) \cdots f_{\mu_2, \Sigma}(x_{140})L(μ1​,μ2​,Σ)=fμ1​,Σ​(x1​)fμ1​,Σ​(x2​)⋯fμ1​,Σ​(x79​)×fμ2​,Σ​(x80​)fμ2​,Σ​(x81​)⋯fμ2​,Σ​(x140​)

      • μ1\mu_1μ1​ 是 Class 1 中 x 的均值，μ2\mu_2μ2​ 是 Class 2 中 x 的均值

      • Σ=79140Σ1+61140Σ2\Sigma = \frac {79}{140} \Sigma_1 + \frac {61}{140} \Sigma_2Σ=14079​Σ1​+14061​Σ2​

      • 像右边的 Model 也称之为 Linear Model（线性模型）

概率模型三个步骤

  • 函数集（模型）
    • 不同的概率分布集合起来
    • input x⟶{P(C1∣x)=P(x∣C1)P(C1)P(x∣C1)P(C1)+P(x∣C2)P(C2)If P(C1∣x)>0.5, output:class 1Otherwise, output:class 2input \, x \longrightarrow \begin{cases} P(C_1|x) = \frac {P(x|C_1)P(C_1)} {P(x|C_1)P(C_1) + P(x|C_2)P(C_2)} \\ If \, P(C_1|x) > 0.5, \, output:class \, 1 \\ Otherwise, \, output: class \, 2 \\ \end{cases}inputx⟶⎩⎨⎧​P(C1​∣x)=P(x∣C1​)P(C1​)+P(x∣C2​)P(C2​)P(x∣C1​)P(C1​)​IfP(C1​∣x)>0.5,output:class1Otherwise,output:class2​
  • 评估函数的好坏
    • 找一个概率分布，能最大化地产生 data 的 likelihood（似然）
  • 找到最好的函数

Posterior Probability（后验概率）

\begin{align*}P(C_1|x) &= \frac {P(x|C_1)P(C_1)} {P(x|C_1)P(C_1) + P(x|C_2)P(C_2)} \\ &= \frac {1}{1 + \frac {P(x|C_2)P(C_2)} {P(x|C_1)P(C_1)}} \\ &= \frac {1}{1 + exp(-z)} &\text{z = \ln \frac {P(x|C_1)P(C_1)}{P(x|C_2)P(C_2)}} \\ &= \sigma(z) &\text{Sigmoid Function} \end{align*}

经过化简，最终得到：z=wTx+bz = w^Tx + bz=wTx+b ，也即 P(C1∣x)=σ(w⋅x+b)P(C_1|x) = \sigma(w \cdot x + b)P(C1​∣x)=σ(w⋅x+b)
