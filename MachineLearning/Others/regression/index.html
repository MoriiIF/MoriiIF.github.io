一、Model（模型）

Model = A set of function $(f_1, f_2, \dots)$

例如，$y = b + w * x$，其中，$b$ 和 $w$ 随意变化。

我们拥有10组 Training Data： $(x1, y_1), \dots, (x{10}, y_{10})$ ，这是真实的数据

二、Goodness of Function（评估函数性能）

  • Model $\longrightarrow$ Goodness of Function「f」 $\longleftarrow$ Training Data

  • 构建损失函数 Loss Function $L$ ：（衡量一组参数的好坏）

    • input — a function
    • output — how bad it is
    • 衡量 Model 中每一个 function 的好坏
    • $\begin{aligned} L(f)=L(w, b)=\sum\limits^{10}_{n=1}(y_n - (b + w * x_n))^2 \end{aligned}$ 【$f$ 由 $b$ 和 $w$ 决定】
    • 上式中，$b + w x_n$ 是基于输入函数所得到的 y 的预估值，$(y_n - (b + w x_n))^2$ 是估测的误差（Estimation error）

三、Best Function（找到最佳函数）

找到最佳函数的公式：$f^* = arg \min\limits_{f} L(f)$

或者也可以写作：$w^, b^ = arg \min\limits{w,b} L(w, b) = arg \min\limits{w, b}\sum\limits^{10}_{n=1}(y_n - (b + w * x_n))^2$
