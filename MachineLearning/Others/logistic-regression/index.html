给定一组 Training Data ：

C1C_1C1​ C1C_1C1​ C2C_2C2​ ⋯\cdots⋯ C1C_1C1​

  • 假定这组数据是基于 fw,b(x)=Pw,b(C1∣x)\displaystyle f_{w,b}(x) = P_{w,b}(C_1|x)fw,b​(x)=Pw,b​(C1​∣x) 产生的

  • 给定一组 www 和 bbb ，产生这组数据的概率是：L(w,b)=fw,b(x1)fw,b(x2)(1−fw,b(x3))⋯fw,b(xN)L(w,b) = f_{w,b}(x_1)f_{w,b}(x_2) \bigl( 1-f_{w,b}(x_3) \bigr) \cdots f_{w,b}(x^N)L(w,b)=fw,b​(x1​)fw,b​(x2​)(1−fw,b​(x3​))⋯fw,b​(xN)

    • 最好的参数 w∗w^*w∗ 和 b∗b^*b∗ 最可能是当 L(w,b)L(w,b)L(w,b) 最大时的 www 和 bbb
    • w∗,b∗=argmax⁡w,bL(w,b)w^*,b^* = arg \max\limits_{w,b}L(w,b)w∗,b∗=argw,bmax​L(w,b) 等同于 w∗,b∗=argmin⁡w,b−ln⁡L(w,b)w^*, b^* = arg \min\limits_{w,b} -\ln L(w,b)w∗,b∗=argw,bmin​−lnL(w,b)

  • 将上面的 Training Data 转化为：（yny_nyn​：1 for class 1, 0 for class 2）

    x1x_1x1​ x2x_2x2​ x3x_3x3​ ⋯\cdots⋯
    y1y_1y1​ = 1 y2=1y_2 = 1y2​=1 y3=0y_3 = 0y3​=0 ⋯\cdots⋯

  • −ln⁡L(w,b)=−ln⁡fw,b(x1)−ln⁡fw,b(x2)−ln⁡(1−fw,b(x3))⋯=∑n−[ynln⁡fw,b(xn)+(1−yn)ln⁡(1−fw,b(xn))]两个伯努利分布的交叉熵(cross entropy)\begin{align*} -\ln L(w,b) &= -\ln f_{w,b}(x_1) - \ln f_{w,b}(x_2) - \ln \bigl(1-f_{w,b}(x_3)\bigr) \cdots \\ &= \sum_n - [y_n\ln f_{w,b}(x_n) + (1-y_n) \ln \bigl(1-f_{w,b}(x_n)\bigr)] \quad \text{两个伯努利分布的交叉熵(cross entropy)} \end{align*} −lnL(w,b)​=−lnfw,b​(x1​)−lnfw,b​(x2​)−ln(1−fw,b​(x3​))⋯=n∑​−[yn​lnfw,b​(xn​)+(1−yn​)ln(1−fw,b​(xn​))]两个伯努利分布的交叉熵(cross entropy)​

    Cross Entropy —— 用于衡量一个概率分布和另一个概率分布之间的距离

    Distribution p : p(x=1)=ynp(x = 1) = y_np(x=1)=yn​；p(x=0)=1−ynp(x = 0) = 1 - y_np(x=0)=1−yn​

    Distribution q : q(x=1)=f(xn)q(x = 1) = f(x_n)q(x=1)=f(xn​)；q(x=0)=1−f(xn)q(x = 0) = 1 - f(x_n)q(x=0)=1−f(xn​)

    交叉熵：H(p,q)=∑xp(x)ln⁡(q(x))H(p,q) = \displaystyle\sum_x p(x) \ln \big(q(x) \big)H(p,q)=x∑​p(x)ln(q(x))
